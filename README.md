# Think

Think is a mobile app to help people to find others that have same preferences.

## Installation

Clone project and install packages:

```bash
git clone https://github.com/mam17/think.git
```
```bash
cd think && yarn
```
```bash
cd ios && pod install
cd ..
```


Run your device.
