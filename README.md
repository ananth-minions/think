# Think

Think is a Instagram clone. Full functional components using HOC.
## Installation

Clone project and install packages:

```bash
git clone https://github.com/mam17/think.git
```
```bash
cd think && yarn
```
```bash
cd ios && pod install (iOS)
cd ..
```


Run your device.
